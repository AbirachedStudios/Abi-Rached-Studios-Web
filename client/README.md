# Abi-Rached-Studios-Web

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- [Node.js](https://nodejs.org/) (which includes npm)

### Installation Guide

#### Step 1: Install Node.js and npm

**Node.js**: It is a JavaScript runtime.  
**npm**: It is the Node.js package manager that will allow you to install project dependencies.

##### Download and Install Node.js

Go to the official [Node.js download page](https://nodejs.org/) and download the latest recommended version (LTS - Long Term Support).

Follow the installation instructions for your operating system:

- **Windows**:
  - Download the .msi installer.
  - Run the installer and follow the instructions.

- **macOS**:
  - Download the .pkg installer.
  - Run the installer and follow the instructions.

- **Linux**:
  - Use your distribution's package manager. For example, on Ubuntu:

    ```bash
    sudo apt update
    sudo apt install nodejs npm
    ```

##### Verify the Installation

Open a terminal and run the following commands to verify that Node.js and npm are installed correctly:

```bash
node -v
npm -v
```
You should see the installed versions of Node.js and npm.

Step 2: Clone the Repository
Open a terminal and clone the repository to your local machine using the following command:

```bash
git clone https://github.com/AbirachedStudios/Abi-Rached-Studios-Web.git
```
Navigate to the project directory:
```bash
cd Abi-Rached-Studios-Web
```
Step 3: Install Project Dependencies
Inside the project directory, install the dependencies using npm:
```bash
npm install
```
This command will read the package.json file and download all necessary dependencies for the project.

Step 4: Run the Project
Now that all dependencies are installed, you can run the project in development mode by executing the following command:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
This will start the Next.js development server. Open your browser and navigate to http://localhost:3000 to see the project in action.

You can start editing the page by modifying app/page.js. The page auto-updates as you edit the file.

Additional Useful Commands
Build the project for production:
```bash
npm run build
```
Start the server in production mode:
```bash
npm run start
```
Run the linter to check the code:
```bash
npm run lint
```
